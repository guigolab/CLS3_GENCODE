## Transcript and loci lists corresponding to the TSSs sets generated.
The TSSs sets are naturally computed from transcripts, therefore the classification in embryo, adult, placenta and common has beed done at transcript level.
Here you find the list of transcript IDs as extracted from the aforementioned sets.

For the loci, also in this folder, a further round of classification was needed as transcripts from the same locus can be associated to any of the aforementioned category. For this reason, if transcripts from a locus are spread across different categories, the locus is assigned to "common".
